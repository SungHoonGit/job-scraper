# 채용공고 자동 검색 모듈

Python + requests/BeautifulSoup 기반 채용공고 자동 수집기.

## Architecture

```
job_scraper.py          # 메인 실행기 (--dry-run / --rebuild)
config.json             # 설정 (검색어/경력/스케줄/사이트별 활성화)
extractors/
  ├── base.py           # 추상 클래스 + 공통 유틸 (fetch, normalize_url, career_code 등)
  ├── jobkorea.py       # 잡코리아 (서버사이드 HTML + ld+json)
  ├── saramin.py        # 사람인 (검색 결과 파싱)
  ├── wanted.py         # 원티드 (Chaos API)
  ├── jumpit.py         # [중단] API 변경
  ├── incruit.py        # [중단] URL 구조 변경
  └── remember.py       # [중단] 도메인 변경
setup.sh                # 한방 설치 (pip + config + launchd 자동 등록)
daily/                  # 수집 결과 (일자별)
history.md              # 최근 N일 통합 뷰
```

## 동작 현황 (2026-06-21)

| 사이트 | 방식 | 상태 | 상세 |
|--------|------|------|------|
| 잡코리아 | 서버사이드 HTML + ld+json | ✅ 활성 | 자유 검색어, detail+기업정보 2회 추가요청, company_info 옵션 |
| 사람인 | 검색 결과 HTML 파싱 | ✅ 활성 | category_code로 직무 설정 (84=웹개발, 235=Java), detail은 SPA |
| 원티드 | Chaos API | ✅ 활성 | `/api/chaos/navigation/v1/results`, 전체 개발공고 → 제목 필터 |
| 점핏 | - | ❌ 중단 | 사람인에 인수 후 SPA 전환, API 응답 HTML |
| 인크루트 | - | ❌ 중단 | URL 구조 완전 변경, 기존 searchjob.asp 404 |
| 리멤버 | - | ❌ 중단 | 도메인 `rememberapp.co.kr`로 변경, API 경로 미확인 |

## 검색 흐름

1. 각 사이트 extractor가 `config.search.query` 등의 설정으로 공고 목록 조회
2. 메인 루프에서 `include_titles` / `exclude_titles` 로 제목 필터
3. 마감일 지난 공고 제외 (`exclude_closed`)
4. `career_from`/`career_to` 설정 시 경력 연차 범위로 post-fetch 필터링
5. `daily/YYYY-MM-DD.md` 에 append + `history.md` 재빌드

## 설정 주요 항목 (config.json)

| 항목 | 설명 |
|------|------|
| `search.query` | 검색어 (자유 텍스트) |
| `search.career_level` | 경력/신입/전체 |
| `search.career_from` | 최소 경력 연차 (예: 2=2년차 이상). 생략 시 필터 안 함 |
| `search.career_to` | 최대 경력 연차 (예: 5=5년차 이하). 생략 시 상한 없음 |
| `search.exclude_closed` | 마감 제외 여부 |
| `include_titles` | 제목 필터 (빈 배열=필터 없음) |
| `exclude_titles` | 제목 제외 키워드 |
| `schedule.enabled/hour/minute` | launchd 자동 실행 설정 |
| `retention_days` | daily 파일 보관 일수 |
| `sites.*.category_code` | 사람인 직무 카테고리 |
| `notification` | 신규 공고 발견 시 macOS 알림 |

## 조사 필요

### 점핏 (jumpit.co.kr)
- 사람인 자회사로 편입 (`jumpit.saramin.co.kr`)
- SPA 전환으로 브라우저 API 확인 필요
- 힌트: 브라우저 개발자도구 Network 탭에서 검색 시 호출되는 XHR 찾기

### 인크루트 (incruit.com)
- 기존 `job.incruit.com/jobdb/list/searchjob.asp` 404
- 사이트는 활성 (ASP.NET 기반)
- 메인 페이지에서 검색해보고 현재 URL 구조 파악 필요

### 리멤버 (remember.co.kr → career.rememberapp.co.kr)
- 도메인 변경: `remember.co.kr` DNS 미응답
- 현재 서비스: `career.rememberapp.co.kr`
- SPA 기반, 브라우저 개발자도구로 API 찾기 필요
